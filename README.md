# Pizzeria Demo

Demo en Astro de una tienda online para pizzeria con UX tipo app de delivery, pensada para mostrar una experiencia realista dentro de TienditaWeb.

## Incluye

- Header sticky con carrito y badge animado
- Navegacion horizontal por categorias con estado activo
- Cards mobile-first con selector de tamano para pizzas
- Drawer de carrito con cantidades, total y pedido por WhatsApp
- Promo destacada, datos del local y mapa mock visual

## Scripts

```bash
pnpm dev
pnpm build
pnpm check
```
